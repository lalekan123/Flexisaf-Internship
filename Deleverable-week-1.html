HTML Tags


HTML tags are keywords present on a web page that define how your web browser must format and display the contents. Such as:


HTML Tag:

The html tag represents the root of an HTML document. The html tag is the container for all other HTML elements (except for the <DOCTYPE> tag).

It's tag: <html>......</html>


Body Tag:

The body tag defines the document's body. The body element contains all the contents of an HTML document, such as headings, paragraphs, images, hyperlinks, etc.
Note: There can only be one body element in an HTML document.

It's tag: <body>......</body>


Heading Tags:

HTML header tags are used to differentiate the headings (h1) and sub-headings (h2-h6) of a page from the rest of the content. Example:

First Heading

It's tag: <h1>......</h1>

Second Heading

It's tag: <h2>......</h2>


Paragraph Tag:

This tag defines a paragraph of text. Examples:

My first paragraph.

This is another paragraph.

It's tag: <p>......</p>


Div Tag:

The div tag in HTML is a container that is used to group other HTML elements together and apply styles to them as a unit. It doesn't have any semantic meaning and is used purely for layout purposes.

It's tag: <div>......</div>


Underline Tag:

The header element represents a container for introductory content or a set of navigational links. Example

This text is underlined

It's tag: <u>......</u>


Bold Tag:

The b HTML element is used to draw the reader's attention to the element's contents, which are not otherwise granted special importance. Example:

This text is bold

It's tag: <b>......</b>


Italics Tag:

The i tag defines a part of text in an alternate voice or mood. The content inside is typically displayed in italic. Example:

This text is in italics

It's tag: <i>......</i>


Span Tag:

The span tag is used to be able to make changes to a certain piece of text. Example:

I can edit this sentence by underlining "edit this".

It's tag: <span>......</span>


Anchor/Link Tag:

An anchor is a piece of text which marks the beginning and/or the end of a hypertext link. The text between the opening tag and the closing tag is either the start or destination (or both) of a link. Example:

Flexisaf.com

It's tag: <a....>......</a>




Self Closing Tags


This is a kind of HTML tag that does not need to be closed manually by its closing tag, which means it does not have a separate closing tag. Such as:


Image Tag:

This tag is used to embed an image in a web page. Example:

It's tag: <img ....>


Input Tag:

The input tag specifies an input field where the user can enter data. The input element is the most important form element. Example:

Text Input


Number Input



It's tag: <input ....>


hr Tag:

This tag is used to draw a thematic line accross in the webpage. Example:

This is the line

It's tag: <hr>


br Tag:

This tag is used to create a line break in text, effectively starting a new line without starting a new paragraph. Example:

Line would break here
The text continues here

It's tag: <br>




Semantic Html Elements


Semantic HTML elements are those that clearly describe their meaning in a human and machine-readable way. Such as:


Header Element:

The header element represents a container for introductory content or a set of navigational links.
It's tag: <header>......</header>


Nav Element:

The nav tag defines a set of navigation links. Notice that NOT all links of a document should be inside a nav element.
It's tag: <nav>......</nav>


Section Element:

The section tag defines a section in a document.
It's tag: <section>......</section>


Article Element:

The article HTML element represents a self-contained composition in a document, page, application, or site, which is intended to be independently distributable or reusable.
It's tag: <article>......</article>


Footer Element:

The footer element represents a container usually at the end of a whole section. You can have several footer elements in one document.
It's tag: <footer>......</footer>
